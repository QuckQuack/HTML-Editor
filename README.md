# HTML-Editor
This is a GUI editor to modify and create checklists and export them in html. It is intended to be accessed via a github pages site which can be found at https://quckquack.github.io/HTML-Editor/

As a disclaimer this is not entirely vibe coded nor is it entirely human written. I wrote the majority of the code base for it in Adobe Dreamweaver, and then used claude.ai to help me debug it and fix a few items within it.
